# uipro

CLI de **scaffolding de composants UI assisté par IA**. Initialisé pour Claude via :

```bash
uipro init --ai claude
```

- **Installé** : `~/.nvm/versions/node/v22.14.0/bin/uipro` (global npm).
- **Rôle dans AkadIA** : volet **design/web** — générer/poser des composants UI cohérents quand AkadIA aura une interface.

> [!note] `init --ai claude` configure le projet pour que l'assistant Claude pilote le scaffolding UI. À lancer dans le dossier du futur front d'AkadIA, pas dans `wiki/`.

S'articule avec [[aura.build]] (catalogue de sections) et [[Awwwards]] (références visuelles). Voir [[Boîte à outils AkadIA]].
