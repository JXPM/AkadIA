# Awwwards

Vitrine de **sites web primés**, filtrable par technique/style (ex. `parallax`).

- **URL** : `https://www.awwwards.com/websites/parallax/`
- **Rôle dans AkadIA** : **référence design haut de gamme** — animations, parallax, mise en page, micro-interactions. À explorer via [[Playwright]] et capturer les exemples retenus dans `assets/`.

> [!tip] Pour chaque site retenu : noter ce qui est réutilisable pour AkadIA (effet, rythme, structure) dans une page concept `2x - …`, relier à [[aura.build]] (composant équivalent) et [[uipro]] (implémentation).

Voir [[Boîte à outils AkadIA]].
